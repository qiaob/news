# AI 日报 — 2026-07-24

## 今日头条：白宫指控 Moonshot 用禁运 GB300 芯片、"工业级蒸馏" Anthropic Fable 打造 Kimi K3

白宫科技政策办公室（OSTP）主任 Michael Kratsios 公开指控中国 AI 创业公司 Moonshot（月之暗面）通过"大规模隐蔽蒸馏"窃取 Anthropic 旗舰模型 Fable 来训练上周发布的开源模型 Kimi K3，称其搭建了可在多种访问方式间快速切换以规避检测的内部平台；同时指控 Moonshot 拥有并在泰国接入了受美国出口管制的 Nvidia GB300 服务器。Anthropic 此前称发现数百万次未授权查询。Kimi K3 拥有 2.8 万亿参数、100 万 token 上下文，在部分基准上逼近甚至超过 Fable 5 与 GPT-5.6，其表现一度引发全球科技股抛售；不过多位专家（据 TechCrunch）质疑"蒸馏 Fable"并非 K3 变强的主因。

「为什么重要」：模型供给的地缘政治风险正在抬头——若头部开源权重被贴上"侵权/违规"标签，接入这类低成本模型的合规与声誉成本会上升；CMB 这类把推理外包给多家厂商的应用，应把"某供应商因出口管制或知识产权争议而不可用"纳入多供应商冗余与合规评估。

**OpenAI 发布企业级 agent 平台 Presence，主打受管控的语音与聊天客服**

OpenAI 于 7 月 22 日推出 Presence——一套面向企业部署 AI agent 的托管平台，覆盖语音与聊天渠道，内置公司政策/SOP、权限控制、人工升级规则与上线前仿真测试；其 Codex 组件基于 GPT-5.6，会审查交互并提出改进，但需人工审批后才上线。OpenAI 用它支撑自家英文电话客服，数周内即达到人工质检标准、解决约 75% 来电，BBVA Mexico、SoftBank 已落地。这使企业 agent 平台一个月内出现 OpenAI Presence、Google Gemini Enterprise、Meta Business Agent 等多家正面竞争。

「为什么重要」：Presence 把"合规护栏 + 权限 + 仿真评测 + 自动改进"打包成产品范式，正是把 LLM 安全接入生产环境的标准动作；对做用户沟通、客服与安全审核的交友平台，这套"策略 + 权限 + 人审 + 评测闭环"是自建 agent 时可直接借鉴的工程模板。

**Alphabet 上调 2026 年资本开支至 1950–2050 亿美元，云收入 +82% 但自由现金流转负**

Alphabet 于 7 月 22 日盘后财报将全年 capex 指引从 1800–1900 亿美元上调至 1950–2050 亿美元，理由是 AI 需求持续超过供给；Google 云收入同比大涨 82% 至 248 亿美元、超预期，但当季自由现金流转负，市场担忧 AI 投入的回收周期，次日股价跌约 7%，与同日重挫的 Tesla 一起拖累纳指。

「为什么重要」：上游持续加码算力，长期利好 Gemini 与 Flash 档模型的供给与价格竞争力；但"capex 先行、现金流承压"意味着头部厂商终将把成本压力向 API 定价传导，选型时要关注谁能把 AI 投入转化为可持续的低价供给。
