# AI 日报 — 2026-07-14

## 今日头条：Google 与 Microsoft 联手多家巨头，推共用"智能体接入标准"围剿 Anthropic 的 MCP

据 The Information 报道，Google、Microsoft、Salesforce、Snowflake 与 ServiceNow 已达成一致，将共同支持一套连接 AI 智能体与企业软件的技术标准，矛头直指 Anthropic 与 OpenAI。过去约 18 个月里，Anthropic 主导的 MCP（Model Context Protocol）已事实上成为智能体接入企业数据与工具的默认标准，而这几家掌握了全球绝大多数企业数据所在软件的巨头，显然不愿把地基建在竞争对手的协议之上。若它们推出统一替代标准，任何部署智能体的企业都将获得一个"由 Google 与 Microsoft 背书"的选项。

「为什么重要」：这场竞争的核心已经从"谁的模型更强"转向"谁掌握智能体接入企业系统的管道与标准"。标准之争决定生态归属，Anthropic 凭 MCP 建立的先发护城河正遭遇迄今最强的集团式反扑。

**Musk 与 Altman 因 Apple 起诉 OpenAI 再度公开互撕**

Apple 于 7 月 10 日起诉 OpenAI 窃取硬件商业机密，并称 OpenAI 目前雇用了 400 多名前 Apple 员工，导致其设计团队被迫重建。事件让 Elon Musk 与 Sam Altman 的宿怨在 X 上再次升级：Musk 重拾"Scam Altman"的旧称嘲讽对方，Altman 则回击 Musk 兜售"短命的太空数据中心"来忽悠公开市场投资者。两人围绕各自 AI 与航天资产的 IPO 与主导权明争暗斗。

「为什么重要」：口水战背后是 xAI 与 OpenAI 对 AI 未来主导权、算力与资本叙事的正面争夺；Apple 诉讼则可能重塑行业挖角与竞业边界的判例风向。

**中国开源权重模型以极低价格逼近前沿能力，华尔街研究"官方认证"其性价比**

多方分析指出，中国开源权重模型已能以开放价格提供前沿模型 80%–90% 的能力，价格差距悬殊：DeepSeek 的输出价格约为每百万 token 0.44 美元，而 GPT-5.6 Sol 高达 30 美元。对于占企业 token 消耗大头的"高频、低风险"工作负载，这一差距几乎无法忽视。当华尔街最保守的投行也在客户研究中把这笔账正式算清，"中国模型只是猎奇"的时代宣告结束。

「为什么重要」：成本结构正成为企业选型的决定性变量，开源权重模型对闭源前沿厂商的定价权构成实质性挤压，全球 AI 竞争格局与商业模式面临重估。

**Anthropic 扩大 Project Glasswing：向 15 国、逾 150 家关键基础设施机构开放 Claude Mythos**

Anthropic 大幅扩展其网络安全计划 Project Glasswing，将受限的前沿模型 Claude Mythos Preview 的访问权从最初约 50 家合作机构扩大到覆盖 15 个国家的 150 余家组织，涵盖公用事业、医疗系统及资源不足的开源项目，获准者包括 Okta、Samsung、SK 海力士、NATO 与欧盟网络安全局 ENISA。该模型已发现数千个高危漏洞，波及各大操作系统与浏览器。Anthropic 表示不会向公众发布 Mythos 级模型。

「为什么重要」：这是迄今规模最大的"用前沿 AI 主动加固关键软件"的实践，既展示了顶级模型在攻防两端的威力，也凸显了前沿能力受控释放与治理的现实路径。

**OpenAI 推出全双工语音模型 GPT-Live，可"边听边说边思考"**

OpenAI 正在全球范围铺开新一代语音模型 GPT-Live（含 GPT-Live-1 与 GPT-Live-1 mini），取代原有的 ChatGPT 语音体验。其采用全双工架构，能同时收听与说话，用户可随时打断、要求放慢或让它安静；对话中还能实时翻译、边聊边搜网页，并把复杂任务委派给后台的 GPT-5.5。付费用户默认使用 GPT-Live-1，免费用户使用 mini 版，后续也将开放 API。

「为什么重要」：自然、可打断的实时语音正把 AI 交互从"轮流问答"推向"连续对话"，语音有望成为智能体落地消费与企业场景的关键入口。
